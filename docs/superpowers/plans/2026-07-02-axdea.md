# AXdea Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 아이디어를 DiceBear 캐릭터로 띄워 물리 캔버스에서 튕겨다니게 하고, 클릭 시 아이디어 카드를 여는 익명 아이디어 보드(AXdea)를 만든다.

**Architecture:** 순수 HTML/CSS/JS 정적 웹앱. Supabase(anon key)로 브라우저에서 직접 CRUD. 경량 자체 물리 엔진(rAF 루프)으로 캐릭터 이동/충돌/던지기 처리. 순수 로직(물리 step, 열람 판정, 아바타 배정)은 node assert로 검증. UI/통합은 로컬 서버 + 브라우저로 확인.

**Tech Stack:** HTML5, CSS(변수 기반 라이트/다크), Vanilla JS, supabase-js(CDN), DiceBear HTTP API, GitHub Pages.

**Testing note:** rhythm과 동일하게 빌드/테스트 프레임워크 없음. 순수 함수는 `js/pure.js`로 분리해 `node test/pure.test.mjs`로 검증(무프레임워크, assert). UI는 브라우저 수동 확인.

- [ ] **Step 1: `supabase.sql` 작성**

```sql
create table if not exists ideas (
  id uuid primary key default gen_random_uuid(),
  title text not null,
  body text default '',
  category text default 'etc',
  color text default '#FFD6A5',
  avatar_style text not null,
  avatar_seed text not null,
  author text not null,
  created_at timestamptz default now()
);

create table if not exists comments (
  id uuid primary key default gen_random_uuid(),
  idea_id uuid references ideas(id) on delete cascade,
  author text not null,
  body text not null,
  created_at timestamptz default now()
);

alter table ideas enable row level security;
alter table comments enable row level security;

create policy "anon read ideas"   on ideas   for select using (true);
create policy "anon write ideas"  on ideas   for insert with check (true);
create policy "anon delete ideas" on ideas   for delete using (true);
create policy "anon read comments"  on comments for select using (true);
create policy "anon write comments" on comments for insert with check (true);
```

```js
import assert from "node:assert";
import { isRevealer, pickAvatar, stepBody, resolveWall } from "../js/pure.js";

// 열람 판정: 정확히 "박찬영"만 true
assert.equal(isRevealer("박찬영"), true);

// 벽 반사: 왼쪽 벽 넘으면 vx 부호 반전 + 위치 보정
const w = resolveWall({ x: -5, y: 100, vx: -4, vy: 0, r: 20 }, 800, 600, 0.8);
assert.ok(w.vx > 0); assert.ok(w.x >= 20);

console.log("PASS pure.test");
```

- [ ] **Step 3: `js/pure.js` 구현**

```js
export function isRevealer(name){ return (name||"").trim() === "박찬영"; }

// 문자열 seed → 안정적 해시
function hash(s){ let h=2166136261; for(let i=0;i<s.length;i++){ h^=s.charCodeAt(i); h=Math.imul(h,16777619);} return h>>>0; }

const STYLES = ["fun-emoji","bottts","adventurer","thumbs","open-peeps","big-smile"];
export function pickAvatar(seed){
  const h = hash(String(seed));
  return { style: STYLES[h % STYLES.length], seed: String(seed) };
}

export function stepBody(b, dt){ return { ...b, x: b.x + b.vx*dt, y: b.y + b.vy*dt }; }

export function resolveWall(b, W, H, damp){
  let { x,y,vx,vy,r } = b;
  if (x < r){ x = r; vx = Math.abs(vx)*damp; }
  if (x > W-r){ x = W-r; vx = -Math.abs(vx)*damp; }
  if (y < r){ y = r; vy = Math.abs(vy)*damp; }
  if (y > H-r){ y = H-r; vy = -Math.abs(vy)*damp; }
  return { ...b, x,y,vx,vy };
}

// 두 원 충돌 시 부드러운 반발(속도 교환 근사)
export function resolveCollision(a, b){
  const dx=b.x-a.x, dy=b.y-a.y; const dist=Math.hypot(dx,dy)||0.001;
  const overlap=(a.r+b.r)-dist;
  if(overlap<=0) return [a,b];
  const nx=dx/dist, ny=dy/dist, push=overlap/2;
  const A={...a, x:a.x-nx*push, y:a.y-ny*push, vx:a.vx-nx*0.5, vy:a.vy-ny*0.5};
  const B={...b, x:b.x+nx*push, y:b.y+ny*push, vx:b.vx+nx*0.5, vy:b.vy+ny*0.5};
  return [A,B];
}
```
